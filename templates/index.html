DZtools

Web-based data visualization and interpretation for detrital zircon geochronology data



Updates

Generating mixed samples and unmixing them to determine provenance is underway. Eventually, the web version of DZstats will be phased out in favor of an intuitive GUI that puts all the DZtools in one place.


How-To

DZstats is the only fully-functional app so far developed. Click the link in the banner to try it out. With it, you can generate and export plots such as KDE, PDP, CDF, and MDS. You can also make matrices like similarity, cross-correlation, likeness, etc. Down this test data file, head over to DZstats, upload the file, and select the plots or matrices that you want to generate. You can use the download buttons to download plots in SVG, PDF, PNG, or EPS formats. The matrices can be downloaded as XLS, XLSX, and CSV files.

Note that this website is currently hosted on Render's free tier, which limits its processing capabilities significantly. We are working with only 0.1 GB of RAM. Because of this, it would be wise to only generate one graph at a time. If you overload the machine, the web browser will display something along the lines of "This page has stopped working." If you receive this error, don't worry. Just go to the home page again. Once we receive funding, we can upgrade to a tier with more processing power.
